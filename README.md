# Notepad

A minimal, distraction-free writing surface for the web.

Notepad is a single-screen text editor designed for focus. It lives in your browser and behaves like a digital scratchpad—open a tab, start typing, and close it when you're done. Your work is always there when you come back.

### Privacy First
There is no backend, no cloud sync, and no user tracking. Everything you type is saved directly to your browser's `localStorage`. Your data never leaves your device.

### Features
- **Distraction-free UI**: Deep dark mode inspired by Linear.
- **Markdown Support**: Type `#` for headings, `-` for lists, `**` for bold.
- **Rich Media**: Paste or drag-and-drop images directly into the editor.
- **Auto-save**: Content persists across tabs and restarts.

### Running Locally
This project uses [Bun](https://bun.sh).

```bash
# Install dependencies
bun install

# Start development server
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
